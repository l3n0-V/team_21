# team_21 - Norwegian Language Learning App

HCAI Semester project

## Overview

A mobile language learning app designed to help students learn Norwegian through real-world interactions. The app encourages users to practice Norwegian by talking to strangers through daily, weekly, and monthly challenges.

## Features

- **Daily Challenges**: Short, quick tasks to practice Norwegian in everyday situations
- **Weekly Challenges**: More involved challenges that build on daily practice
- **Monthly Challenges**: Comprehensive goals that test overall language proficiency
- **Progress Tracking**: Monitor your learning journey and completed challenges

## Tech Stack

- **React Native** with **Expo** - Cross-platform mobile development
- **JavaScript/React** - Frontend framework

## Getting Started

### Prerequisites

- Node.js (v20+)
- npm or yarn
- Expo CLI

### Installation

1. Clone the repository:
```bash
git clone https://github.com/l3n0-V/team_21.git
cd team_21
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Run on your device:
- Scan the QR code with the Expo Go app (Android/iOS)
- Or press 'w' to open in web browser

### Running on Specific Platforms

- **Android**: `npm run android`
- **iOS**: `npm run ios` (requires macOS)
- **Web**: `npm run web`
